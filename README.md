# IT1214-Warehouse-Inventory-System1
## Description
This is a Java-based Warehouse Inventory System using ArrayList.
It allows users to manage items efficiently.

## Features
- Add items
- Remove items
- Update quantity
- Search items
- Display all items

## How to Run

javac Warehouse.java
java Warehouse

## Sample Output

1 Add Item
2 Remove Item
3 Update Quantity
4 Search Item
5 Display All Items
6 Exit
